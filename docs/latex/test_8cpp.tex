This file contains a series of test cases to verify the functionality of the \doxylink{class_perceptron}{Perceptron} and \doxylink{class_perceptron_layer}{Perceptron\+Layer} classes. The tests include training and prediction for various logic gates.

Test Cases\+:
\begin{DoxyItemize}
\item \doxylink{class_perceptron}{Perceptron} for INVERT Gate\+: Tests the perceptron\textquotesingle{}s ability to learn the INVERT gate.
\item \doxylink{class_perceptron}{Perceptron} for AND Gate\+: Tests the perceptron\textquotesingle{}s ability to learn the AND gate.
\item \doxylink{class_perceptron}{Perceptron} for OR Gate\+: Tests the perceptron\textquotesingle{}s ability to learn the OR gate.
\item \doxylink{class_perceptron}{Perceptron} for NOR Gate (3 inputs)\+: Tests the perceptron\textquotesingle{}s ability to learn the NOR gate with 3 inputs.
\item \doxylink{class_perceptron}{Perceptron} for 3-\/input Majority Gate\+: Tests the perceptron\textquotesingle{}s ability to learn the 3-\/input Majority gate.
\item \doxylink{class_perceptron_layer}{Perceptron\+Layer} for AND and OR Gates\+: Tests the \doxylink{class_perceptron_layer}{Perceptron\+Layer}\textquotesingle{}s ability to learn the AND and OR gates.
\item \doxylink{class_perceptron_network}{Perceptron\+Network} for the XOR gate with 2 inputs.
\item \doxylink{class_perceptron_network}{Perceptron\+Network} for a half adder.
\end{DoxyItemize}
